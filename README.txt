Project Overview:- This project is part of UDACITY Primer's training. In this project a static website has been hosted using s3 bucket and cloudfront in AWS.

Technologies Used:- Amazon S3 – for static website hosting Amazon CloudFront – for content delivery with public access AWS IAM – for public access control and security policy

Project Highlights:-

Created a bucket named: ruchika-first-bucket
Uploaded and managed website files (HTML, CSS)
Set appropriate bucket permissions and policies via IAM
Configured S3 Bucket to enable static web hosting
Created a CloudFront distribution for low-latency content delivery
Access Website in Web Browser using CloudFront domain name : dkfq8yliz995t.cloudfront.net
Access Website in Web Browser using CloudFront website endpoint URL : http://ruchika-first-bucket.s3-website.us-east-2.amazonaws.com/
Access Website in Web Browser using its S3 object URL: https://ruchika-first-bucket.s3.amazonaws.com/index.html

Purpose:- Sole aim of this project is understanding AWS and hands-on experience in static website hosting.
