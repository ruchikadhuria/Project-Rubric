Project-Rubric
This project is part of the Udacity AWS Cloud Primer training program and focuses on deploying a static website using AWS services. The project demonstrates how to host a website on cloud storage and deliver it globally with high performance using a Content Delivery Network (CDN).
?? Project Overview:-
 In this project, a static website is hosted using Amazon S3 and distributed via Amazon CloudFront to ensure fast, secure, and reliable access across different geographic locations.
??? Technologies & Services Used:- 
Amazon S3 – for hosting static website files Amazon CloudFront – for global content delivery AWS IAM – for access control and security HTML, CSS – for website structure and styling
?? Project Highlights:-
1. Created a bucket named: ruchika-first-bucket
2. Uploaded and managed website files (HTML, CSS)
3. Set appropriate bucket permissions and policies via IAM
4. Configured S3 Bucket to enable static web hosting
5. Created a CloudFront distribution for low-latency content delivery
6. Access Website in Web Browser using CloudFront domain : dkfq8yliz995t.cloudfront.net
7. Access Website in Web Browser using website-endpoint : http://ruchika-first-bucket.s3-website.us-east-2.amazonaws.com/
8. Access Website in Web Browser using its S3 object URL: https://ruchika-first-bucket.s3.amazonaws.com/index.html
?? Key Features:- 
Global content delivery with reduced latency Scalable and highly available architecture Secure access management using IAM policies Cost-effective hosting solution
?? Learning Outcomes:- 
Understanding how CDNs like CloudFront improve performance Hands-on experience with AWS storage and networking services Knowledge of deploying and managing static websites in the cloud Basics of cloud security and permissions
?? Purpose:- 
The purpose of this project is to gain practical experience in deploying and optimizing a static website using AWS cloud infrastructure.
